____________________________________________SETUP_________________________________________________________________________________________				
1.) Start the Apache and MySQL modules using the WAMP controller.
2.) Open the phpMyAdmin and create a database "store". 
3.) Import the store.sql file present in the SQL folder.
4.) Open the www folder in the wamp folder. Copy paste the folder PHP.
5.) Open the browser (chrome), type localhost/PHP/index.php and you should see the index page of the website.
